# **App Name**: Narasimha V C Portfolio

## Core Features:

- Responsive Section Display: Showcase all key portfolio sections (Hero, About Me, Skills, Projects, Education, Contact, Footer) in a clean, modern, and responsive layout that adapts to various screen sizes.
- Dynamic Project Showcase: Display 2-4 sample projects with title, description, tech stack, and links (GitHub/demo). Include subtle hover effects for interactivity.
- Interactive Skills Presentation: Present programming languages, web technologies, and tools using clean cards or visually engaging indicators.
- Contact Form: Allow visitors to send messages via a simple contact form. This will capture input and send it to a predefined endpoint.
- Smooth Scroll Navigation: Enable fluid and smooth scrolling between different portfolio sections from the main navigation.
- Professional Styling & Animation: Implement modern typography, generous spacing, subtle scroll animations, and clean UI to ensure an elegant and professional user experience.

## Style Guidelines:

- Color Palette Rationale: A light, professional aesthetic is achieved through a cool blue-grey tone, promoting readability and a modern feel, with a subtle accent blue adding a touch of digital sophistication. This reflects professionalism and the technological aspect of an aspiring developer, without being overly corporate. This decision aligns with the user's request for a clean, modern, and professional light palette. Primary accent color: #2671CA (a sophisticated blue, conveying professionalism and reliability) (HSL: 220, 70%, 50%). Background color: #F3F6FA (a very light, desaturated blue-gray for a soft, clean base) (HSL: 220, 15%, 95%). Secondary accent color: #66CEDC (a bright, complementary cyan for highlights and interactivity) (HSL: 190, 60%, 65%). Text color: Dark gray or black for optimal readability against the light background. For subtle overlays and hover effects, translucent variations like `rgba(38, 113, 202, 0.1)` for a light blue tint or `rgba(102, 206, 220, 0.15)` for a brighter, airy touch will be used to maintain visual lightness.
- Headlines will use 'Space Grotesk' (sans-serif) for a modern, tech-oriented, and structured feel. Body text will utilize 'Inter' (sans-serif) to ensure high readability and a clean, objective aesthetic, complementing the headline font without overwhelming it.
- Implement minimalist, modern line-art style icons from a consistent icon library (e.g., Lucide or Feather Icons) for social media links, skill representations, and calls-to-action to maintain a clean and professional look.
- The layout will be section-based, utilizing ample whitespace to create an uncluttered, elegant presentation. A fluid grid system will ensure mobile-first responsiveness, adapting seamlessly to various screen sizes. Sections will feature clear headings and well-organized content blocks.
- Incorporate subtle, tasteful animations including smooth scrolling between sections, gentle fade-in effects for content on scroll, and elegant hover transitions on interactive elements like project cards and navigation links, enhancing user engagement without distracting from the content.
